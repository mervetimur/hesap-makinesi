# Hesap Makinesi

Bu depo basit bir hesap makinesi programının kodlarını içerir.
Kodlar codes/code.txt dosyasında bulunuyor.

## Kullanım

1. `codes/code.txt` dosyasındaki kodları kendi Java/IDE ortamınıza kopyalayın.  
2. Çalıştırın ve temel matematiksel işlemleri deneyin: toplama, çıkarma, çarpma, bölme.  

## Amaç

Bu proje, temel Java programlama ve algoritma mantığını öğrenmek isteyenler için örnek bir uygulamadır.
